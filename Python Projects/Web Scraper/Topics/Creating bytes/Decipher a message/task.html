Decipher a message

Let's decipher a little message. The first input line is an encoded message, and the second line is the key integer number. Convert the key to two bytes and sum up its items. Then add the resulting sum to the code point of each character in the message. Finally, print the decoded message.


Sample Input:
HlAdghmcXnt
256

Sample Output:
ImBehindYou



Memory limit: 256 MB
Time limit: 15 seconds
